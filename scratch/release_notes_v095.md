# Release Notes: v0.9.5

We are pleased to announce the release of **v0.9.5** for the Jenkins-2026 stack! This release introduces programmatic environment protection configuration guides in the project documentation.

---

## 🚀 What's New in v0.9.5

### 1. Environment Protection CLI Automation Documentation
* Documented step-by-step programmatic setup instructions in `README.md`.
* Explains how to query a user's GitHub ID and apply environment reviewer constraints using the `gh api` CLI command, supplementing the manual web UI setup instructions.
