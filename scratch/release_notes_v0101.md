# Release Notes: v0.10.1

This release expands the **Visual & Multimedia Repo Guide** by introducing Spanish-language versions of our key architectural and operational walkthroughs.

---

## 🚀 What's New in v0.10.1

### 1. Spanish Multimedia Guide (via NotebookLM)
We have added Spanish versions of our multimedia assets to make the Jenkins-2026 stack more accessible:
*   **Fábrica DevOps jenkins-2026 (Español)**: A video demonstration in Spanish covering the core features of the repository.
*   **Infraestructura completa por veinte céntimos (Español)**: A Spanish audio deep-dive into the cost-optimized ephemeral infrastructure model.

### 2. README Enhancements
*   All multimedia resources are now explicitly labeled by language (English/Spanish) to ensure clear navigation for all users.
